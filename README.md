Purpose
The planner helps students:
- Plan ahead and choose courses for each quarter.
- Understand which **prerequisites** are needed before taking advanced classes.
- View courses organized by **1000-, 2000-, 3000-, 4000-, and 5000-levels**.
- Track progress toward graduation more efficiently.

---

Features
- Lists when each course is typically offered (Fall, Winter, Spring, Summer).
- Highlights prerequisite chains between courses.
- Organizes classes by level for clear academic progression.
- Designed for **Seattle University CS majors** to plan the next school year or entire degree path.
